# 凶灵传奇

这是太阳神三国杀的一个武将扩展包，以某个未实际作品化的脑洞《凶灵传奇》为主题而作，是对原版三国杀的部分武将进行疯狂加强后的结果。共包含如下8名武将：

1. 凶灵司马懿（天灵、狼残、凶魂）

2. 凶灵魏文长（狂血、凶络）

3. 凶灵孙尚香（闺阵、弓机、凶姿）

4. 凶灵天公将（天道、伏雷、凶魄）

5. 凶灵古恶来（暴戟、凶骨）

6. 凶灵曹子桓（阎罗、狱罚、凶势）

7. 凶灵贾文和（绝计、毒谋、凶容）

8. 凶灵钟士季（操控、凶运）
